Action

The action is pressing DELETE on the keyboard as long as the requirements don't mention something else.

Test 1: checked

Requirements:
  • At least one checkbox is checked.
Expected result:
  • The page stays open.
  • The testSuiteResult is "Selected items > 0"

Test 2: not checked (happy path)

Requirements:
  • No checkbox is checked.
Expected result:
  • The page closes.
  • The testSuiteResult is "Closing"

Test 3: modifier pressed

Requirements:
  • A modifier (Alt, Shift, Control, Meta, OS, AltGraph) is pressed together with Delete
Expected result:
  • The page stays open.
  • The testSuiteResult is "Modifier pressed"

Test 4a: input fix (input)

Requirements:
  • No checkboxes are checked
  • The text input is focused
Expected result:
  • The page stays open.
  • The testSuiteResult is "Input fix"

Test 4b: input fix (textarea)

Requirements:
  • No checkboxes are checked
  • The textarea is focused
Expected result:
  • The page stays open.
  • The testSuiteResult is "Input fix"

Test 5: contenteditable fix

Requirements:
  • No checkboxes are checked
  • The contenteditable is focused
Expected result:
  • The page stays open.
  • A character is removed (if that's the default behaviour of the browser)
  • The testSuiteResult is "Contenteditable fix"
